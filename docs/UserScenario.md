# 마이링크 (MyLink) - 사용자 시나리오 (User Scenario)

## 1. 방문자 (Visitor): 마이링크 페이지 방문 과정

- 사용자는 **소유자의 다양한 채널 정보를 확인하기 위해** 소유자가 공유한 고유 URL(`서비스도메인.com/사용자ID`)에 **접속한다.**
- 사용자는 **원하는 특정 콘텐츠나 웹사이트로 이동하기 위해** 화면에 나열된 여러 **링크 블록 중 하나를 클릭한다.** (현재 창에서 해당 웹사이트로 바로 이동)
- 사용자는 **자신만의 멀티 링크 페이지를 개설하기 위해** 페이지 하단에 위치한 **'마이링크 만들기' 브랜드 로고 및 버튼을 클릭한다.**

## 2. 소유자 (Owner): 링크 추가/수정/삭제 과정

### 2.1 링크 추가 (Add)
- 사용자는 **자신의 새로운 웹사이트 주소를 등록하기 위해** 관리자 페이지 메인 화면에서 **'링크 추가' 버튼을 클릭한다.**
- 사용자는 **방문자에게 보여줄 정보와 이동 경로를 설정하기 위해** 제목(Title)과 실제 주소(URL)를 **직접 입력란에 작성하고 '추가'를 클릭한다.**
- 사용자는 **등록한 사이트의 시각적 식별이 잘 되는지 확인하기 위해** URL에서 자동으로 추출되어 표시되는 **파비콘(Favicon) 썸네일을 확인한다.**

### 2.2 링크 수정 (Edit)
- 사용자는 **기존에 등록한 링크의 제목이나 주소를 변경하기 위해** 해당 링크 블록 우측에 있는 **'수정' 버튼을 클릭한다.**
- 사용자는 **잘못된 정보를 바로잡거나 새로운 정보를 반영하기 위해** 활성화된 입력란의 텍스트를 **직접 수정하고 '저장' 버튼을 클릭한다.**

### 2.3 링크 삭제 (Delete)
- 사용자는 **더 이상 유효하지 않거나 노출하고 싶지 않은 링크를 제거하기 위해** 해당 링크 블록의 **'삭제' 버튼을 클릭한다.**
- 사용자는 **실수로 데이터를 날리는 것을 방지하기 위해** 화면에 나타난 기본 **삭제 확인 팝업창(또는 알림창)에서 '확인'을 클릭하여 영구 삭제한다.**
